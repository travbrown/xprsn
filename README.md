# XPRSV

XPRSV (pronounced "Expressive") is a creative portfolio by Trav Brown — a space dedicated to the art of seeing.

**Live site:** [xprsv.vercel.app](https://xprsv.vercel.app)

## Pages

- **Home** — Landing page with YouTube background video and about section
- **Films** — Video gallery with lightbox playback for short films and commercials
- **Photography** — Photo gallery
- **Contact** — Contact form

## Tech Stack

- React
- React Router
- Material UI
- Deployed on Vercel

## Development

```bash
npm install
npm start
```

Runs on [http://localhost:3000](http://localhost:3000).

```bash
npm run build
```

Creates a production build in `build/`.
